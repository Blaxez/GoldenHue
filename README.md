GoldenHue Pro - Advanced Palette Generator
GoldenHue Pro Logo

Overview
GoldenHue Pro is a sophisticated web application designed for designers, developers, and artists to effortlessly generate, explore, and refine harmonious color palettes. It stands out with its intuitive interface, powerful color theory algorithms, and real-time visual feedback.

Features
Diverse Harmony Modes: Generate palettes based on various color harmony rules (e.g., Monochromatic, Analogous, Complementary, Triadic, Tetradic) to ensure aesthetically pleasing combinations.
Interactive Live UI Previews: See your chosen palettes applied instantly across different UI elements (buttons, cards, text, backgrounds) in both light and dark modes, allowing for immediate visual assessment.
Advanced Customization: Fine-tune individual colors within a generated palette, adjust saturation, brightness, and hue, and lock specific colors to preserve them during regeneration.
Modern Neumorphic Design: Experience a sleek, convex neumorphic user interface that provides a soft, tactile feel, enhancing user interaction and visual appeal.
Theme Toggle: Seamlessly switch between light and dark themes to test palette visibility and accessibility under different lighting conditions.
Export Options: Easily export generated palettes in various formats (e.g., CSS variables, SCSS variables, JSON) for quick integration into projects.
Installation
To get a local copy up and running, follow these simple steps.

Clone the repository:
git clone https://github.com/Blaxez/GoldenHue.git
Navigate to the project directory:
cd GoldenHue
Open index.html in your web browser to start using the application.
Usage
Select a harmony mode from the available options.
Adjust the base color using the color picker.
Observe the generated palette and live UI previews.
Toggle between light and dark themes to test accessibility.
Export your favorite palettes for use in your projects.
